* [Docker compose](https://www.docker.com/)

<a name="run-project"/>

## Run project

1. Clone the repo

       $ git clone https://github.com/yovasx2/bmi-calculator

2. Move into it

       $ cd bmi-calculator

3. Build with docker

       $ docker-compose build

3. Up all containers

       $ docker-compose up

5. Visit localhost:4200
